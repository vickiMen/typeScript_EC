Strong VS. Weak typing

  • strong/weak typing is about how strictly types are distinguished
  • JS follows weak typing, with the ability to "cast" a value of any type to another type
  • JavaScript coercion rules
      • (some results may surprise you)
      Example:
      const str = 5 + 'hello' // '5hello' - perfectly OK in JS

    Dynamic VS. Static

  • Duck typing is the ability of a language to determine types at runtime
  • JavaScript follows this duck typing principle - it assigns types to variables at runtime, based on the interpeted value.
      • Example:
      const length = 12
      const num = length(54)
      const num = length(34)

    TypeScript

  • An open source language from Microsoft
  • A superset of JavaScript that compiles to plain JavaScript by the browser
  • Since JavaScript code is TypeScript code you can start off with JavaScript and just add some types here and there
  • Flexibility comes with a price! if you are building a medium- to large-scale application, writing and managing JavaScript code could lead to many bugs...

    • Installing

    npm install -g typescript

    Compiling

    tsc source.ts
